# OpenMall 变更日志

本文档记录 OpenMall 项目的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，版本号遵循语义化版本。

## [10.3.0-SNAPSHOT] - 2026-06-30

### 新增 (Added)
- 项目基础框架搭建（Spring Boot 3.5.10 + JDK 21 + Maven 多模块）
- 公共模块 (openmall-common)：核心工具类、领域实体、安全认证、Web 配置
- 前台商城模块 (openmall-portal)：商品浏览、购物车、订单、用户中心
- 商家后台模块 (openmall-merchant)：商品管理、订单管理、店铺管理、优惠券
- 平台管理模块 (openmall-platform)：用户管理、商家审核、分类品牌管理
- 数据库设计：14 张核心业务表
- Thymeleaf 模板引擎 + Bootstrap 5 前端界面
- Spring Security 认证授权
- Redis 缓存与会话管理
- JDK 21 Virtual Threads 支持
- Docker 部署支持
- 完整文档体系

### 技术栈
- Spring Boot 3.5.10
- Spring Security 6.2.0
- Spring Data JPA
- JDK 21 (Virtual Threads)
- Maven 4.0.0+
- MySQL 8.0+
- Redis 6.0+
- Thymeleaf 3.1.1
- Bootstrap 5
- Lombok 1.18.42
- Hutool 5.8.25

### 核心功能
- 用户注册/登录（买家、商家、管理员）
- 商品 SPU/SKU 管理
- 商品分类与品牌管理
- 购物车管理
- 订单生命周期管理
- 收货地址管理
- 商品收藏与浏览记录
- 商家入驻审核
- 优惠券管理
- 数据统计控制台

## 版本号说明

- **MAJOR**：不兼容的 API 修改
- **MINOR**：向下兼容的功能新增
- **PATCH**：向下兼容的问题修复

## 变更类型

- `Added`：新增功能
- `Changed`：功能变更
- `Deprecated`：即将废弃
- `Removed`：已移除
- `Fixed`：Bug 修复
- `Security`：安全修复
